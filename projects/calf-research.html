Calf Research

Our research focuses on improving dairy calf management using advanced machine learning techniques like YOLOv8 for body weight prediction and diarrhea detection. This work integrates precision livestock farming with cutting-edge technologies.

Key highlights of this project include:

  • Using depth images for accurate body weight prediction.
  • Automated health monitoring through computer vision.
  • Integration of genetic analysis with calf health data.
